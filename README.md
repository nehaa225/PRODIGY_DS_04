# PRODIGY_DS_04

## Sentiment Analysis on Social Media Data

### 📌 Task Overview

This project is part of the **Prodigy InfoTech Data Science Internship – Task 4**.
The objective of this task is to perform **Sentiment Analysis** on social media text data to determine whether the sentiment expressed in the text is **positive, negative, or neutral**.

Sentiment analysis helps organizations understand **public opinion, customer feedback, and trends** by analyzing textual data from online platforms.

---

### 📊 Dataset

The dataset contains **text data collected from social media**, including comments or posts that express opinions about various topics.

Key attributes in the dataset include:

* **Text / Comment** – Social media message
* **Sentiment** – Label indicating the emotion (Positive, Negative, Neutral)

---

### ⚙️ Technologies Used

* **Python**
* **Jupyter Notebook**
* **Pandas** – Data manipulation
* **TextBlob** – Sentiment analysis
* **Matplotlib / Seaborn** – Data visualization

---

### 🔍 Project Workflow

1. **Data Loading**

   * Import the dataset using Pandas.

2. **Data Cleaning**

   * Handle missing values
   * Remove duplicates

3. **Sentiment Analysis**

   * Apply **TextBlob** to determine sentiment polarity.
   * Classify text into:

     * Positive
     * Negative
     * Neutral

4. **Data Visualization**

   * Visualize sentiment distribution using charts.

---

### 📈 Results

The analysis helps in understanding how people feel about certain topics by categorizing the sentiment of social media posts. The visualizations provide a clear overview of the **overall sentiment distribution**.
